• 基本的な心構え
    • 他国と大きく変わりありませんし、あえて長々と記載しません。他者への礼節を欠くような行為は避けてください。
    • 途中放棄、突然死はゲームを壊し、他参加者に多大な迷惑をかけることになるため、他サイトよりも厳しい措置としています。
  • 能力行使
    • 一部の役職を除き、処理順で処理される時点で死亡している場合、能力は行使されません。
      • 死亡していても処理される能力の場合、役職説明に記載しています。（例. 人狼、罠師、爆弾魔）
    • 処理される時点で死亡して能力が行使されなくても、足音はセットした通りに響きます。
  • 突然死
    • 村の設定で突然死を無しにしている場合、突然死しません。
    • 突然死ありの場合、進行中の日付更新のタイミングで、投票していなかった生存者が突然死します。
      • 突然死を回避するために発言する必要はありません。
    • 一度でも突然死した場合、自己蘇生する役職であっても、後追い処理の中で再度死亡するため、朝時点で必ず死亡状態となります。
    • 突然死の対象となる参加者は、「本日まだ投票していない者は、・・・」という表現でログの最後に表示されます。
    • 突然死者の投票は無効票となります。
    • 突然死者のセットしていた足音については、セットした通りに響きます。
    • 突然死者はエピローグ以外で一切の発言ができなくなり、以後別の村に入村もできなくなります。
      • 突然死した方は国主まで直接ご連絡ください。事故や災害など、やむを得ない事情により突然死してしまったと判断した場合、入村制限を解除します。
  • 投票
    • 2日目から投票による処刑が始まります。
    • 突然死無しの場合：日付更新時のセット先は自投票です。
    • 突然死ありの場合：日付更新時のセット先はなしです。（投票しないまま更新時刻を迎えると突然死します。）
  • アンカー

    • >>=1 の形式で共鳴発言へのアンカーが付けられます。
    • >>+1 の形式で墓下発言へのアンカーが付けられます。
    • >>@1 の形式で見学発言へのアンカーが付けられます。
    • >>*1 の形式で囁きへのアンカーが付けられます。
    • >>_1 の形式で念話へのアンカーが付けられます。
    • >>?1 の形式で恋人発言へのアンカーが付けられます。
    • >>#1 の形式で村建て発言へのアンカーが付けられます。
    • >>a1 の形式でアクションへのアンカーが付けられます。
    • >>-1 の形式で独り言へのアンカーが付けられます。（エピローグ後のみ）
    • >>s1 の形式で秘話へのアンカーが付けられます。（エピローグ後のみ）
    • いずれのアンカーも自分が見ることのできない発言はクリックしてもアンカー先を表示できません。
  • 役職配分
    • 村建て時に変更可能です。デフォルトは以下のようになっています。（ダミーは必ず村人になります）
    • 08-09人：ダ1 狼2 賢1導1 村3-4
    • 10-11人：ダ1 狼2狂1 賢1導1狩1 村3-4
    • 12-13人：ダ1 狼3狂1 賢1導1狩1 村4-5
    • 14-15人：ダ1 狼3魔1 狐1 賢1導1狩1 霊5-6
    • 16-17人：ダ1 狼3魔1 狐1 賢1導1狩1共2 霊5-6
    • 18-20人：ダ1 狼4魔1 狐1 賢1導1狩1共2 霊6-8
    • 闇鍋編成にした場合、1日目に役職が決定した後も編成の内訳は公開されません。
    • ただし、以下は保証されています。
      • 人狼カウント役職が1名以上
      • 人狼カウント役職の人数+1 ＜ 人間カウント役職の人数
      • 恋人と同棲者はそれぞれ偶数
      • 妖狐系なしで背徳者・陰陽師がいる、パン屋なしで餡麺麭者がいるなど、即死する役職が存在しない
  • 闇鍋編成
    • 村設定で闇鍋編成となっている場合、陣営や役職ごとに細かく配分設定ができ、以下のように編成が決定されます。
    • 役職ごとに最少人数が1名以上で設定されている場合、先に確保します。
    • 次に、陣営の最少人数が1名以上で設定されている場合、先に確保します。
    • 残りは、一人ずつ陣営を抽選 → 陣営内の役職を抽選します。
    • 陣営、役職ともに、配分の値が高いほど抽選率が上がります。
    • 陣営の抽選率＝[陣営の配分 / 全陣営の配分の合計]です。
    • 役職の抽選率＝[役職の配分 / 陣営内の役職の配分の合計]です。
    • 割り振った結果以下の条件を1つでも満たすと50回まで再抽選しますが、50回で決まらなかった場合1日目開始時にエラーとなります。
      • 2日目にPP発生
      • 人狼カウントの最少最多設定を満たしていない
      • 恋人や同棲者が奇数
      • 妖狐系なしで背徳者・陰陽師がいる、パン屋なしで餡麺麭者がいるなど、即死する役職がある
  • 役職希望
    • ダミーキャラ（2日目に人狼に襲撃される人）は必ず村人となります。
    • 村建て時に役職希望を無効にした場合、全員がおまかせ扱いになります。
    • 役職希望ありの場合、第1希望、第2希望それぞれ役職や陣営などを指定することができます。
    • おまかせ系の希望は、役職指定での希望よりも後で役職が割り振られます。
    • おまかせ（村人陣営、人狼陣営、恋人陣営、妖狐陣営、愉快犯陣営）は、陣営に属する役職の希望となります。
    • おまかせ（人外）は、人狼陣営＋妖狐陣営＋愉快犯陣営の役職の希望となります。
    • おまかせ（足音職）は、人狼系、狂人系、占い師系、狩人、妖狐の希望となります。
    • おまかせ（役職窓あり）は、人狼系、C国狂人、共鳴者、恋人、同棲者の希望となります。おまかせ（役職窓なし）は逆です。
    • 1日目に移行する際、以下の処理で配役が決定されます。
      • 1. 人数に応じて構成が変化するため、以下のように自動で希望が変更されます。
        • ダミーを除いて村人が0名で、霊能者が1名以上存在する場合は、村人→霊能者に変更
        • 霊能者が0名で、村人がダミーを除いて1名以上存在する場合は、霊能者→村人に変更
        • 占い師と賢者は、存在しない場合、存在する側に変更
        • 狂人とC国狂人と狂信者と魔神官は、存在しない場合、C狂>狂信>魔>狂の順で探して存在する役職に変更
        • 上記以外で構成上存在しない役職を希望していた場合はおまかせに変更
      • 2. 上記自動変更後、第1希望がおまかせ、第2希望がおまかせ以外の場合は、第1希望と第2希望を入れ替え
      • 3. 第1希望のうち、役職指定で希望した人を割り振り
      • 4. 第1希望のうち、おまかせ系で希望した人を割り振り
      • 5. 第2希望のうち、役職指定で希望した人を割り振り
      • 6. 第2希望のうち、おまかせ系で希望した人を割り振り
      • 7. ここまでで割り振られなかった人（おまかせもしくは希望が叶わなかった人）を割り振り
    • G国にあった「初参加者は強制村人希望」「30%の確率でおまかせになる」仕様はありません。
  • 見学
    • 村建て時に見学入村を可能にしていると見学入村できます。
    • 見学で入村すると、1日目から死亡扱いとなり、墓下＋見学者同士でのみ会話が可能となります。（地上からは発言は見えません）
    • 見学入村できる最大人数はキャラチップにより異なります。[キャラチップ人数 - 村定員]人まで見学入村が可能です。
    • 例えば、キャラセットで人狼BBSを選択し、定員を17名にしていた場合は32-17=15名まで見学入村が可能です。
  • プロデューサー
    • 村建て時にプロデューサー機能をありにしていると、村建てがプロデューサー機能を持ち、通常入村ができなくなります。
    • プロデューサー機能
      • 参加者がどのプレイヤーかを把握できます。
      • 独り言を含め、全ての発言・メッセージが見られます。
  • 発言
    • 1発言につき400文字20行まで発言できます。
    • 村建て発言は、1発言につき1000文字40行まで発言できます。
    • 発言制限が設定されている村では、発言できる最大文字数と1日のうちに発言できる回数が制限されます。
    • ランダム機能については、その他 - ランダム機能を参照してください。
  • アクション
    • 1発言につき400文字まで発言できます。
    • 発言制限が設定されている村では、発言できる最大文字数と1日のうちに発言できる回数が制限されます。
    • ランダム機能については、その他 - ランダム機能を参照してください。
  • 名前変更・簡易メモ
    • 名前は40文字まで、略称は1文字、簡易メモは20文字まで入力することができます。
    • 推理に繋がる内容を記載しないでください。
    • 簡易メモは状況 - 参加者欄にのみ表示されます。
  • コミット
    • 設定でコミットありになっている村では、ダミー以外の生存者全員がコミットすることで次の日に進めることができます。
    • 誰がコミットしているかは周りからわかりません。現在コミットしている人数のみ表示されます。
    • コミットは取り下げることもできます。ただし、他全員がコミット済みの場合はすぐに次の日に進むため取り下げることはできません。
    • コミットにより次の日に進んだ場合、次の日がその分長くなります。（つまり、次の更新日時は本来の更新日時のままです。）
  • 転生
    • 転生者やトラックの能力などによりランダムな役職に転生する場合、選ばれる役職は以下のように決定されます。
      • 恋人、同棲者など、ペアで編成されることが前提となる役職は選択されません。
      • 村建てオプションで転生先役職が「全ての役職」になっている村では、上記ペア役職以外の全ての役職から選択されます。
      • 村建てオプションで転生先役職が「編成に存在する役職」になっている村では、以下のように選択されます。
        • 固定編成の場合は、編成に含まれる役職から選択されます（例えば17人村で、16人編成に含まれて17人編成に含まれない役職などは選択されません）。
        • 闇鍋編成の場合は、「配分」が0でない役職から等確率でランダムに選択されます（最多数は無視されます）。
    • 後述する「役職変化」についても参照ください。
  • 役職変化
    • 転生者、申し子、トラック、死霊術師など、役職が変化した場合の処理やルールについて記載します。
    • 役職が変化すると、勝敗判定の陣営も役職に応じて変更されます。ただし、恋絆や狐憑きなど、陣営変化ステータスが優先されます。
    • 役職変化しても恋絆や狐憑きなどのステータスは削除されません。
    • 役職変化後は、過去の役職のシステムメッセージを見ることはできません。
    • 役職変化前にセットしていた能力は、役職変化後に処理順を迎える場合、不発します。
      • 例. パン屋が強制転生により他役職に変化した場合、パン屋メッセージは表示されません。
    • 役職変更前後のCOや結果報告については、現在の役職で参照できる情報のみCO及び結果発表を可とします。
      • 導師→人狼に変化した場合、導師のときに知った情報は報告してはいけません。
      • 探偵→導師に変化した場合、導師として参照できる情報はCOして報告できますが、探偵のときに知った情報は報告してはいけません。
      • 魔神官→人狼→導師に変化した場合、魔神官のときに知った情報も導師COして報告できます（導師でも参照できるため）。
      • 役職窓など、通常発言以外であれば何を話しても問題ありません。
  • 流れ
    • プロローグ
      • 更新時間までに最少開始人数が揃っていた場合、更新時間に1日目に遷移します。
      • 24時間ログイン状態でのアクセスがない参加者は自動で村を去ります。
    • 1日目
      • 部屋および役職が各個に割り当てられます。
      • 見学者が墓下へ移動し、生存者と会話できなくなります。（オプションで変更可能です）
      • 投票および処刑はありません。
      • 人狼の襲撃対象はダミーキャラ固定になります。襲撃担当者および経路は選択できます。
    • 2日目以降
      • 能力行使した内容に応じて部屋に足音が響きます。
      • 処刑するための投票先を選択できるようにます。
      • 人狼が襲撃対象を選択できるようになります。
      • 日付更新時、勝利条件に当てはまった場合はエピローグに遷移します。
