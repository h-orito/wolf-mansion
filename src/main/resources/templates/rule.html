人狼の基本ルール

  • 通常の人狼のルールを熟知していることが参加の前提となるため、基本ルールは省略します。

人狼館の事件簿村ルール

  • 白ログでの推理発言、まとめ行為、CO禁止です。（グレラン村）
    • 役職持ち、騙り含め全てのCOおよび結果報告が禁止となります。
  • ただし、導師のみCOおよび霊能結果を白ログで報告することができます。（結果騙りもOKとします。）
    • 導師も白ログでの推理やまとめ行為は禁止です。
    • 当然ながら、役職者騙りも禁止です。
    • 墓下と地上で会話ができる村設定の場合は、死亡後の導師CO及び結果の報告は禁止です。
    • これらの禁止行為は通常村でのルールとなります。村ごとに定めたルールがある場合はそちらが優先されます。
  • 1日目になると、各参加者に碁盤の目のように並んだ部屋のいずれかが割り振られます。
  • 以下の能力を発動すると、「通過した部屋」に足音が響き、翌日朝に纏めて報告されます。
    • 占い師による占い
    • 狼による襲撃
    • 狩人による護衛
    • 狂人、C国狂人、魔神官、狂信者、妖狐による徘徊（任意）
  • 部屋移動と響く足音について、以下のルールがあります。
    • 行動対象に向かって縦横の最短距離で移動し、通過する部屋に足音が響きます。
      • 曲がる必要がある場合は1回曲がる経路のうちどちらかを選択できます。
      • 部屋が隣接している場合は足音は響きません。
      • 生存者のいない部屋は足音が響きません。
    • 人狼は襲撃者を選択することができます。
      ただし、設定で同一人狼による連続襲撃が不可となっている村では、人狼が複数生存している場合、連続で同じ人狼が襲撃することはできません。
    • 狂人、C国狂人、魔神官、狂信者、妖狐は自分の部屋からに限らず、どの部屋からどの部屋まで徘徊するか自由に選択できます。
      • ただし、曲がることはできません。
      • 通過した部屋だけでなく、徘徊した部屋全てに足音が響きます。
    • 狂人、C国狂人、魔神官、狂信者、妖狐は「徘徊しない」こともできます。
    • 翌日足音が報告される順番はランダムで、部屋番号は若い順です。
    • 翌日に能力者、襲撃担当者が死亡していてもセットした通りに足音が響きます。

詳細ルール

  • 基本的な心構え
    • 他国と大きく変わりありませんし、あえて長々と記載しません。他者への礼節を欠くような行為は避けてください。
    • 途中放棄、突然死については、他国よりも厳しい対処とする可能性があります。
  • 突然死
    • 村の設定で突然死を無しにしている場合、突然死しません。
    • 突然死ありの場合、進行中の日付更新のタイミングで、投票していなかった生存者が突然死します。
      • 他国と異なり、発言する必要はありません。
    • 突然死の対象となる参加者は、「本日まだ投票していない者は、・・・」という表現でログの最後に表示されます。
    • 突然死者の投票は無効票となります。
    • 占い師や賢者が突然死した場合、占いは行われません。（したがって、呪殺も起きません。）
    • 狩人が突然死した場合、護衛は行われません。
    • 人狼が突然死した場合、他に人狼が生存していればセットした通りに襲撃が実施されます。
    • 突然死者のセットしていた足音については、セットした通りに響きます。
      • 占い師、賢者、狩人は能力行使先の初期設定がランダムであるため、響く可能性があります。狂人、C国狂人、魔神官、狂信者、妖狐は初期設定が「なし」のため響きません。
    • 突然死者はエピローグ以外で一切の発言ができなくなり、以後別の村に入村もできなくなります。
      • 突然死した方は国主まで直接ご連絡ください。会話させていただいたのち入村制限解除とします。
      • 連絡いただければ基本的に制限解除しますが、改善が見込めない場合など、解除しない場合があります。
  • 投票
    • 2日目から投票による処刑が始まります。
    • 突然死無しの場合：日付更新時のセット先は自投票です。
    • 突然死ありの場合：日付更新時のセット先はなしです。（投票しないまま更新時刻を迎えると突然死します。）
  • アンカー
    • >>1 の形式で白発言へのアンカーが付けられます。
    • >>=1 の形式で共鳴発言へのアンカーが付けられます。
    • >>+1 の形式で墓下発言へのアンカーが付けられます。
    • >>@1 の形式で見学発言へのアンカーが付けられます。
    • >>*1 の形式で囁きへのアンカーが付けられます。
    • >>-1 の形式で独り言へのアンカーが付けられます。（エピローグ後のみ）
    • いずれのアンカーも自分が見ることのできない発言はクリックしてもアンカー先を表示できません。
  • 役職配分
    • 村建て時に変更可能です。デフォルトは以下のようになっています。（ダミーは必ず村人になります）
    • 08-09人：ダ1 狼2 賢1導1 村3-4
    • 10-11人：ダ1 狼2狂1 賢1導1狩1 村3-4
    • 12-13人：ダ1 狼3狂1 賢1導1狩1 村4-5
    • 14-15人：ダ1 狼3魔1 狐1 賢1導1狩1 霊5-6
    • 16-17人：ダ1 狼3魔1 狐1 賢1導1狩1共2 霊5-6
    • 18-20人：ダ1 狼4魔1 狐1 賢1導1狩1共2 霊6-8
  • 役職希望
    • ダミーキャラ（2日目に人狼に襲撃される人）は必ず村人となります。
    • 村建て時に役職希望を無効にした場合、全員がおまかせ扱いになります。
    • 役職希望ありの場合、第1希望、第2希望それぞれ役職や陣営などを指定することができます。
    • おまかせ系の希望は、役職指定での希望よりも後で役職が割り振られます。
    • おまかせ（村人陣営、人狼陣営）は、陣営に属する役職の希望となります。
    • おまかせ（人外）は、村人陣営以外の役職の希望となります。
    • おまかせ（足音職）は、人狼、狂人系、占い師系、狩人、妖狐の希望となります。
    • おまかせ（役職窓あり）は、人狼、C国狂人、共鳴者の希望となります。
    • 1日目に移行する際、以下の処理で配役が決定されます。
      • 1. 人数に応じて構成が変化するため、以下のように自動で希望が変更されます。
        • ダミーを除いて村人が0名で、霊能者が1名以上存在する場合は、村人→霊能者に変更
        • 霊能者が0名で、村人がダミーを除いて1名以上存在する場合は、霊能者→村人に変更
        • 占い師と賢者は、存在する側に変更
        • 狂人とC国狂人と狂信者と魔神官は、存在する側に変更
        • 上記以外で構成上存在しない役職を希望していた場合はおまかせに変更
      • 2. 上記自動変更後、第1希望がおまかせ、第2希望がおまかせ以外の場合は、第1希望と第2希望を入れ替え
      • 3. 第1希望のうち、役職指定で希望した人を割り振り
      • 4. 第1希望のうち、おまかせ系で希望した人を割り振り
      • 5. 第2希望のうち、役職指定で希望した人を割り振り
      • 6. 第2希望のうち、おまかせ系で希望した人を割り振り
      • 7. ここまでで割り振られなかった人（おまかせもしくは希望が叶わなかった人）を割り振り
    • G国にあった「初参加者は強制村人希望」「30%の確率でおまかせになる」仕様はありません。
  • 見学
    • 村建て時に見学入村を可能にしていると見学入村できます。
    • 見学で入村すると、1日目から死亡扱いとなり、墓下＋見学者同士でのみ会話が可能となります。（地上からは発言は見えません）
    • 見学入村できる最大人数はキャラチップにより異なります。[キャラチップ人数 - 村定員]人まで見学入村が可能です。
    • 例えば、キャラセットで人狼BBSを選択し、定員を17名にしていた場合は32-17=15名まで見学入村が可能です。
  • 発言
    • 1発言につき400文字20行まで発言できます。
    • 村建て発言は、1発言につき1000文字40行まで発言できます。
    • 発言数制限は未実装です。
    • ランダム機能については、その他 - ランダム機能を参照してください。
  • コミット
    • 設定でコミットありになっている村では、ダミー以外の生存者全員がコミットすることで次の日に進めることができます。
    • 誰がコミットしているかは周りからわかりません。現在コミットしている人数のみ表示されます。
    • コミットは取り下げることもできます。ただし、他全員がコミット済みの場合はすぐに次の日に進むため取り下げることはできません。
  • 流れ
    • プロローグ
      • 更新時間までに最低開始人数が揃っていた場合、更新時間に1日目に遷移します。
      • 24時間ログイン状態でのアクセスがない参加者は自動で村を去ります。
    • 1日目
      • 部屋および役職が各個に割り当てられます。
      • 見学者が墓下へ移動し、生存者と会話できなくなります。（オプションで変更可能です）
      • 投票および処刑はありません。
      • 人狼の襲撃対象はダミーキャラ固定になります。襲撃担当者および経路は選択できます。
      • 占い師、賢者は占い対象を選択できます。
      • 狂人、魔神官、C国狂人、狂信者、妖狐は徘徊する部屋を選択できます。
      • 狩人は能力行使できません。
    • 2日目以降
      • 能力行使した内容に応じて部屋に足音が響きます。
      • 処刑するための投票先を選択できるようにます。
      • 人狼が襲撃対象を選択できるようになります。
      • 狩人が護衛対象を選択できるようになります。
      • 日付更新時、勝利条件に当てはまった場合はエピローグに遷移します。

役職説明

  • 村人
    • 特別な能力はありません。
  • 占い師
    • 毎晩一人を占い、人間か人狼かを知ることができます。
      占うと、自分の部屋から対象の部屋に向かう際通過した部屋に足音が響きます。
    • 処刑により死亡した場合、足音は響きますが、占いは行われません。
    • 襲撃により死亡した場合、占いは実行され、足音も響きます。
    • 「対象なし」はありません。必ず占い対象を選択することになります。
    • 日付更新時のセット先はランダムです。
    • 妖狐を占った場合「人間」と判定されます。呪殺による死亡なのか襲撃による死亡なのかは判別できません。
  • 賢者
    • 毎晩一人を占い、その人の役職を知ることができます。
      占うと、自分の部屋から対象の部屋に向かう際通過した部屋に足音が響きます。
    • 処刑により死亡した場合、足音は響きますが、占いは行われません。
    • 襲撃により死亡した場合、占いは実行され、足音も響きます。
    • 「対象なし」はありません。必ず占い対象を選択することになります。
    • 日付更新時のセット先はランダムです。
  • 人狼
    • 人狼とC国狂人同士にしか聞こえない会話が可能です。
      また、毎晩一人を襲撃することができます。
      襲撃すると、襲撃者の部屋から対象の部屋に向かう際通過した部屋に足音が響きます。
    • 処刑により死亡した場合も襲撃担当者が変わることなく襲撃は実行されます（襲撃なしにはなりません）。（生存している人狼が0名になった場合は襲撃は実行されません）
    • 設定で同一人狼による連続襲撃不可となっている村では、人狼が複数生存している場合、2日連続で同じ人狼が襲撃することはできません。
    • 日付更新時の襲撃者およびセット先はランダムです。
    • 「襲撃なし」はありません。必ず襲撃対象を選択することになります。
    • 襲撃失敗の場合、護衛なのか妖狐襲撃なのかは判別できません。
    • 「●●！今日がお前の命日だ！」の発言者は生存している人狼からランダムで選ばれるため、襲撃セット時の襲撃者と異なることがあります。（襲撃者が死亡していることがあるため連動させていません）
  • 狩人
    • 毎晩一人を人狼の襲撃から守ることができます。
      護衛すると、自分の部屋から対象の部屋に向かう際通過した部屋に足音が響きます。
    • 処刑により死亡した場合、足音は響きますが、護衛は行われません。
    • 襲撃により死亡した場合、護衛は実行され、足音も響きます。
    • 「対象なし」はありません。必ず護衛対象を選択することになります。
    • 日付更新時のセット先はランダムです。
    • 護衛手応えはありません。
    • 設定で連続護衛不可となっている村では、同一人物を2日連続で護衛することができません。
    • 1日目はセットできません。
  • 霊能者
    • 処刑された・突然死した人物が人間であったか人狼であったかを知ることができます。（妖狐は人間判定です）
  • 導師
    • 処刑された・突然死した人物の役職を知ることができます。
  • 狂人
    • 毎晩任意の部屋からその直線上の部屋に向かって徘徊し、徘徊した部屋に足音を響かせることが可能です。
    • 「足音なし」が可能です。
    • 日付更新時の足音セットは「なし」です。
  • C国狂人
    • 人狼とC国狂人同士にしか聞こえない会話が可能です。
      また、毎晩任意の部屋からその直線上の部屋に向かって徘徊し、徘徊した部屋に足音を響かせることが可能です。
    • 「足音なし」が可能です。
    • 日付更新時の足音セットは「なし」です。
  • 魔神官
    • 処刑された・突然死した人物の役職を知ることができます。
      また、毎晩任意の部屋からその直線上の部屋に向かって徘徊し、徘徊した部屋に足音を響かせることが可能です。
    • 「足音なし」が可能です。
    • 日付更新時の足音セットは「なし」です。
  • 狂信者
    • 人狼が誰なのかを知ることができます。（人狼からは自分が狂信者であることはわかりません）
    • 毎晩任意の部屋からその直線上の部屋に向かって徘徊し、徘徊した部屋に足音を響かせることが可能です。
    • 「足音なし」が可能です。
    • 日付更新時の足音セットは「なし」です。
  • 共鳴者
    • 共鳴者同士にしか聞こえない会話が可能です。
  • 妖狐
    • 人狼に襲撃されても死亡しませんが、占われると死亡します。
      また、毎晩任意の部屋からその直線上の部屋に向かって徘徊し、徘徊した部屋に足音を響かせることが可能です。
    • 「足音なし」が可能です。
    • 日付更新時の足音セットは「なし」です。
    • 処理順は処刑→占い→襲撃となるため、占い師が処刑された場合、妖狐に占いをセットしていても妖狐は死亡しません。逆に、占い師が襲撃された場合でも妖狐に占いをセットしていた場合は妖狐は死亡します。
