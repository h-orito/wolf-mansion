• 恋絆
    • 後追い対象が死亡すると、自身も後追死します。
    • 恋絆が付与されている人同士にしか聞こえない恋人会話が可能です。
    • Bさんへの恋絆がAさんに付与されていて、Aさんへの恋絆がBさんに付与されていない場合、Bさんは後追死しませんし、Bさんは恋人会話を使用することもできません。（いわゆる片想い状態）
    • 自身の役職にかかわらず、勝利条件が恋人陣営のものに上書きされます。
      • あなたは[03農]農夫 ヤコブに恋しています。
    • [02長]村長 ヴァルターは、絆に引きずられるように[03農]農夫 ヤコブの後を追った。
  • 狐憑き
    • 自身の役職にかかわらず、勝利条件が妖狐陣営のものに上書きされます。
    • ただし、恋絆が付与されている場合は勝利条件が恋人陣営のままとなります（例. 誑狐に誑かされた後に求愛者に求愛された）。
    • 妖狐系役職が誰であるかを知ることができます（恋絆が付与されていても知ることができます）。
    • 念話を聞くことができます（話せるのは仙狐と念力が付与されている人のみです）。
    • あなたを誑かした妖狐以外からは、あなたが妖狐陣営に与する者だとはわかりません。
    • 妖狐系役職が全滅すると、後追いで死亡します（恋絆が付与されている場合は後追いしません）。
      • あなたは妖狐に与するものとなりました。
    • [02長]村長 ヴァルターは、妖狐の後を追い、いなくなってしまった。
  • 狂気
    • 自身の役職にかかわらず、勝利条件が人狼陣営のものに上書きされます。
    • ただし、恋絆や狐憑きが付与されている場合はそのステータスの勝利条件が優先されます（例. 教唆者に唆された後に求愛者に求愛された）。
    • 人狼を含む他の人からは、あなたが人狼陣営に与する者だとはわかりません。
      • あなたは狂気を宿しています。
  • 信念
    • 自身の役職にかかわらず、勝利条件が村人陣営のものに上書きされます。
    • ただし、恋絆/狐憑き/狂気が付与されている場合はそのステータスの勝利条件が優先されます（例. 牧師に説かれた後に求愛者に求愛された）。
    • 他の人からは、あなたが村人陣営に与する者だとはわかりません。
      • あなたは平和を望んでいます。
  • 保険
    • このステータスが付与された状態で死亡すると、蘇生できます。
    • 蘇生するとこのステータスは解除されます（2つ以上付与されていても、全て解除されます）。
    • 処理順は蘇生系の最後となるため、他の手段で蘇生した場合はこのステータスは解除されません。
    • 蘇生後に後追い処理があるため、後追い対象となっている場合は復活直後に後追死します。
      • あなたは保険を勧められました。
    • 今からでも入れる保険があるんですか！？
      不思議なことに、[02長] 村長 ヴァルターが生き返った。
  • 不敬
    • このステータスが付与された状態で投票されると、被投票数が2倍になります。
    • 2倍の計算は、市長/弁護士/冷凍者などの加減算処理後に行われます。
    • 死亡するとこのステータスは解除されます（強制転生でも解除されます）。
    • 付与・解除・被投票いずれもメッセージは表示されません（被投票数は2倍になっています）。
  • 呪縛符
    • このステータスが付与された状態で占われると、死亡（呪殺）します。
    • 端的にいうと、一時的に妖狐状態になります。
    • 妖狐同様、管狐に占われた場合は呪殺されません。
    • 呪殺されるとこのステータスは解除されます（2つ以上付与されていても、全て解除されます）。
    • 突然死など、占われた時点で既に死亡している場合は、ステータスは解除されません。
    • 妖狐など、もともと呪殺される役職の場合も、呪殺されるとこのステータスは解除されます。
    • 呪狼占いや臭狼調査、反呪符占いなどで逆呪殺された際もこのステータスは解除されます。
    • 反呪符と共存できます。
    • 付与・解除・呪殺いずれの際もメッセージは表示されません。
  • 反呪符
    • このステータスが付与され、生存した状態で占われると、占った側が死亡（呪殺）します。
    • 端的にいうと、一時的に呪狼状態になります。
    • 反呪符状態の探偵が臭狼を調査しても、臭狼は死亡しません。
    • 探偵が反呪符状態の役職（臭狼以外）を調査しても、探偵は死亡しません。
    • 占いの逆呪殺により死亡させるとこのステータスは解除されます（2つ以上付与されていても、全て解除されます）。
    • 複数の占い系役職が同時に反呪符状態の1名を占った場合、最初の1名が死亡します（占い処理順はランダム）。
    • 占われた時点で既に死亡している場合は、占った側は死亡しませんし、ステータスも解除されません。
    • 呪狼など、既に逆呪殺能力を持っている場合も、呪殺により死亡させるとこのステータスは解除されます。
    • 反呪符状態の臭狼を探偵が調査した場合は、呪殺により死亡させますが、占いではないためこのステータスは解除されません。
    • 呪縛符と共存できます。
    • 付与・解除・呪殺いずれの際もメッセージは表示されません。
  • 念力
    • 念話を話すことができます。
    • 1日で効果が切れ、自動的に除去されます。
